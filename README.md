# 33-公交车司机管理系统

技术：
基于 B/S 架构 SpringBoot+MySQL+vue+elementui

环境：
Idea mysql maven jdk1.8  node



#### **用户端功能**

1.首页:展示车辆信息及车辆位置和线路信息
2.模块:车辆信息及车辆位置和线路信息
3.公告、论坛
4.在线留言

5.个人中心:修改个人信息



#### 司机端功能

1.首页、个人中心、修改密码
2.司机、用户管理
3.排班查看
4.打卡功能
5.请假功能
6.申报车辆状态



#### 管理端功能

1.首页、个人中心、修改密码
2.司机、用户管理
3.司机排班管理
4.车辆信息以及车辆位置和线路信息管理
5.论坛管理
6.在线留言:回复用户的留言和增删改查
7.轮播图管理
8.公告
